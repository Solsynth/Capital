# 用户协议

欢迎使用 Solsynth 产品和服务。本用户协议（以下简称“本协议”）构成了您（以下简称“用户”或“您”）与 Solsynth LLC（以下简称“我们”或“Solsynth”）之间具有法律约束力的协议，适用于所有 Solsynth 产品和服务（以下简称“服务”），包括但不限于 Solar Network、Solarpass 等。除非特定产品另有明确声明的附加条款，否则本协议为所有服务的规则框架。

通过访问、使用我们的服务或创建账户，即表示您已阅读、理解并同意本协议及其后续更新。如果您不同意本协议的任何条款，请立即停止使用我们的服务并注销账户。

若您代表组织或实体使用服务，您保证您有权代表该组织或实体同意本协议，并确保该组织或实体遵守本协议。

## 定义和用词声明

为确保清晰，本协议中使用以下关键词汇具有特定含义：

- **可以**：表示允许但非强制性的操作，用户或我们可自行选择是否执行。
- **应该**：表示推荐或预期进行的操作，但无强制义务；不执行不视为违反本协议。
- **必须**：表示强制性要求，用户或我们必须执行；违反视为违反本协议。
- **不得**：表示禁止行为，与“必须”相反；执行视为违反本协议。

其他定义：
- **用户**：指任何访问、使用我们的服务或拥有 Solsynth 账户的个人、组织或实体，包括但不限于自然人、法人或机器人账户。
- **服务**：指 Solsynth 提供的任何产品、平台、API、工具或功能，包括 Solar Network 等。
- **账户**：指用户在 Solsynth 系统中的注册身份，如 Solarpass 账户。
- **用户生成内容（UGC）**：指用户上传、发布或通过服务创建的任何内容，包括文本、图像、视频、音频、文件等。

### 账户管理和安全

1. 用户负责维护账户的安全，包括密码和访问令牌。用户**不得**分享账户凭证。
2. 若发现账户被未经授权使用，用户**应该**立即通知我们。我们**可以**协助调查，但不对任何损失负责。
3. 一个自然人**只能**注册和拥有一个 Solarpass 账户。我们有权删除或合并多余账户，并保留相关数据。
4. 用户**不得**转让、出售或出租账户。发现此类行为，我们将立即终止账户。
5. 机器人账户：通过开发者门户创建的机器人账户不受多账户限制，但必须遵守 [Solar Network 开发者协议](/terms/solar-network-dev-zh)。

### 处罚措施（适用于 Solar Network 等服务）

1. 违反本协议时，用户可能收到以下处罚，视严重程度而定：
   - **警告 (Strike)**：无实际影响，但记录违规。警告在 180 天无进一步违规后自动撤销。若在警告期内再犯，处罚升级。
   - **停权 (Suspension)**：分为“完全停权”和“部分停权”。
     - 完全停权：用户无法访问任何服务，包括登录 Solarpass。通常有固定时限。
     - 部分停权：限制特定功能，如发帖、上传文件等。
   - **禁用 (Ban)**：永久禁用账户及所有 Solsynth 服务访问权。我们保留删除相关数据的权利。
2. 若用户在处罚期间创建新账户绕过处罚，我们将删除新账户，并视情况延长或升级原处罚。
3. 用户有权对处罚提出申诉。通过 lily@solsynth.dev 提交证据，我们将审查并回复。

## 不适当的使用方式

以下行为视为不当使用，我们有权限制功能、发出警告、停权或禁用账户：

1. **恶意利用漏洞**：发现漏洞**必须**上报，而非利用。例如：无限调用签到接口刷取经验。
2. **滥用 API 或资源**：产生垃圾数据、刷屏或浪费服务器资源。例如：自动化脚本每分钟发帖报时。
3. **将正常账户用于自动化**：例如：使用个人账户的令牌开发聊天机器人。推荐使用开发者门户创建机器人账户。
4. **开发违规软件**：违反 [Solar Network 开发者协议](/terms/solar-network-dev-zh)，如修改官方客户端绕过订阅限制。
5. **其他**：包括但不限于骚扰、欺诈、传播恶意软件、绕过内容过滤或任何损害平台生态的行为。

## 用户生成内容 (UGC)

1. UGC 定义：用户通过服务上传或创建的任何内容。
2. 用户对 UGC 承担全部责任。我们不预审 UGC，但保留监控和删除的权利。
3. 用户**必须**确保 UGC 遵守所在国家/地区的法律法规，包括知识产权法、隐私法和反歧视法。
4. **知识产权**：
   - 上传 UGC 即授予我们非独占的、全球性的、免版税许可，用于存储、展示、分发和推广 UGC，以运营服务。
   - 若 UGC 侵犯第三方知识产权，我们将删除内容，并根据重复违规情况发出警告或停权。
5. **禁止内容**：我们**可以**删除以下 UGC 并处罚发布者：
   - 侵犯知识产权、隐私（如开盒、人肉搜索）或名誉。
   - 违反公共道德、本地法律或国际公约（如仇恨言论、暴力威胁、非法内容）。
   - 垃圾信息、刷屏或无意义内容浪费资源。
   - 虚假信息、欺诈或误导性内容。
6. **隐私设置**：我们尊重用户设置（如仅好友可见），并尽力保护数据。但不对意外泄露负责。详见我们的隐私政策。
7. 我们**可以**使用 UGC 进行分析、改进服务，但会匿名化处理。
