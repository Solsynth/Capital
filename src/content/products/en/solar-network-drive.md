## What is Solar Network Drive?

Solar Network Drive is Solar Network's file hosting service. It provides powerful media analysis capabilities and robust infrastructure to support all Solar Network services' file attachment needs.

You can easily upload files in the Solian App and manage your files. Solar Network Drive also provides fixed access links that can be accessed without any tokens based on permission settings. This allows you to use Solar Network Drive as an image hosting service.

Like other services, Solar Network Drive also provides an open API. You can find the relevant documentation at [Suki](https://kb.solsynth.dev/solar-network/developers/file-upload/).
