## What is Solar Network Drive?

Solar Network Drive is a decentralized file hosting solution designed for developers and teams who need reliable, fast file storage without the complexity of traditional cloud services.

### Key Features

- **Drag & Drop Upload** - Simply drag files to upload them instantly
- **Shareable Links** - Generate public or private links for any file
- **API Access** - Integrate file storage into your own applications
- **End-to-End Encryption** - Your files are always secure

### Getting Started

Visit our dashboard at [https://fs.solian.app](https://fs.solian.app) to create an account and start uploading files immediately.

For developers, check out our [GitHub repository](https://github.com/Solsynth/Solian) for API documentation and SDKs.
